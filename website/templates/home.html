We are ChatBuddy!
The Image-Based Chatbot is an AI-powered assistant designed to understand and respond to visual inputs. Users can upload an image and engage in conversation about its contents. Whether it's identifying objects, describing scenes, or answering visual questions, the chatbot delivers intelligent, human-like responses to enhance user interaction with images.
The Conversational Chatbot is a Flask-based AI system integrating OpenAI GPT and OpenAIRouter APIs for intelligent, real-time responses. It features a modular backend with Flask Blueprints, session handling via UUIDs, and a dynamic frontend using HTML, CSS, and JavaScript. Markdown rendering and secure data handling enhance user interaction. .
